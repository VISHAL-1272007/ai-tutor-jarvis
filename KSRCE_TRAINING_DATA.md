## SAMPLE EXAM QUESTIONS BY SUBJECT

### Data Structures (CSE Sem 3)
Q1: Explain the differences between arrays and linked lists with examples.
Q2: Write a program to implement a binary search tree with insert and delete operations.
Q3: Design an algorithm to find the shortest path in a weighted graph using Dijkstra's algorithm.
Q4: Compare bubble sort, quicksort, and mergesort in terms of time and space complexity.

### Database Management (CSE Sem 4)
Q1: Design an ER diagram for a college management system.
Q2: Normalize the following table to BCNF (3NF example given).
Q3: Write SQL queries to find the top 3 departments by student strength.
Q4: Explain ACID properties with examples.

### Discrete Mathematics (CSE/AI&DS)
Q1: Prove using mathematical induction that the sum of first n natural numbers is n(n+1)/2.
Q2: Find the number of paths from (0,0) to (4,5) on a grid (combinatorics).
Q3: Apply Dijkstra's algorithm to find the shortest path in a graph.
Q4: Explain the Tower of Hanoi problem and its recurrence relation.

### Machine Learning (AI&DS Sem 3-4)
Q1: Explain supervised vs unsupervised learning with examples.
Q2: Derive the cost function for linear regression and explain gradient descent.
Q3: Implement K-means clustering algorithm from scratch.
Q4: Compare decision trees vs random forests.

### Engineering Mathematics
Q1: Solve the differential equation dy/dx + 2y = e^(-x) with initial condition y(0) = 1.
Q2: Find eigenvalues and eigenvectors of a 3x3 matrix.
Q3: Apply Laplace transforms to solve a circuit differential equation.
Q4: Perform hypothesis testing: H0: μ = 100, H1: μ ≠ 100, given sample data.

### Operating Systems (CSE Sem 5)
Q1: Explain process scheduling algorithms: FCFS, SJF, Round Robin.
Q2: Solve a deadlock scenario using banker's algorithm.
Q3: Explain virtual memory and page replacement algorithms.
Q4: Design a file allocation strategy for an operating system.
